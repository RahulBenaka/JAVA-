STACKOVERFLOW

IF THE RECURSIVE FUNCTION OF A PROGRAM HAS MORE THAN 10^4 CALLS, THEN IT RESULTS IN STACKOVERFLOW ERROR
HENCE FOR DP ITS BETTER TO USE ITERATIVE METHOD THAN RECURSION AS IT WONT HAVE MORE THAN 10^4 REQUESTS WAITING IN STACK
HENCE IF A PROGRAM HAS MORE THAN 10^4 REQUESTS IN STACK, IT RESULTS IN STACKOVERFLOW
